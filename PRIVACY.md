# 개인정보 처리방침 — 포커스온 (FocusOn)

**최종 갱신: 2026-04-17**

포커스온은 사용자의 개인정보를 수집·저장·전송하지 않습니다. 이 문서는 앱이 사용하는 Android 권한과 그로 인해 접근 가능한 정보의 처리 방식을 투명하게 설명합니다.

---

## 1. 데이터 수집 및 전송

**수집하지 않으며, 외부로 전송하지 않습니다.**

- 분석 SDK, 광고 SDK, 크래시 리포터를 포함하지 않습니다.
- 네트워크 통신 자체가 없습니다. 매니페스트에 `INTERNET` 권한이 선언돼 있지 않습니다.
- 사용자 식별자, 광고 ID, 설치된 앱 목록, 입력 내용, 접속 URL — 모두 **기기 외부로 나가지 않습니다**.

---

## 2. 기기 내부에 저장되는 정보

다음 정보가 앱 전용 샌드박스 (`/data/data/com.focuson.app/`) 안에만 저장됩니다. 타 앱이 접근할 수 없으며, 앱을 삭제하면 함께 삭제됩니다.

| 저장소 | 데이터 |
|---|---|
| Room DB (`focuson.db`) | 차단할 앱 패키지명 · 차단할 사이트 도메인 · 세션 이력 (시작/종료 시간) |
| DataStore Preferences | 진행 중 세션 메타 (ID, 모드, 종료 시각, 엄격모드 여부) · 온보딩 완료 플래그 |

---

## 3. 사용 권한과 그 이유

### 3.1 알림 권한 (`POST_NOTIFICATIONS`)
**용도:** 세션 진행 상태를 상단바에 상시 표시하기 위해.
**수집 정보:** 없음.

### 3.2 다른 앱 위에 표시 (`SYSTEM_ALERT_WINDOW`)
**용도:** 차단된 앱 실행 시 전체화면 안내 오버레이 표시.
**수집 정보:** 없음.

### 3.3 사용정보 접근 (`PACKAGE_USAGE_STATS`)
**용도:** 현재 포그라운드 앱을 감지해 차단 대상인지 판정.
**수집 정보:** 앱 사용 통계를 조회하지만 **저장하거나 전송하지 않습니다**. 판정 직후 즉시 폐기.

### 3.4 접근성 서비스 (`BIND_ACCESSIBILITY_SERVICE`)
**용도:**
1. 앱 전환 이벤트 수신 (차단 대상 앱 감지)
2. 지원 브라우저의 주소창 텍스트 읽기 (차단 사이트 URL 감지)
3. 엄격모드에서 "앱 삭제" · "접근성 설정" · "강제 중지" 화면 진입 감지

**수집 정보:**
- 입력한 텍스트를 **일절 저장/전송하지 않습니다**. 주소창 텍스트만 차단 판정 직후 폐기.
- 비밀번호, 메시지, 검색어 등 일반 텍스트 필드는 아예 접근하지 않습니다.

접근성 서비스 설정 파일(`res/xml/accessibility_service_config.xml`)에서 수신 이벤트를 "창 상태 변경" · "창 내용 변경" · "텍스트 변경"으로만 제한합니다.

### 3.5 설치된 앱 목록 조회 (`QUERY_ALL_PACKAGES`)
**용도:** "차단할 앱 선택" 화면에서 설치된 앱 목록을 보여주기 위해.
**수집 정보:** `PackageManager`에서 목록을 조회하지만 **저장하지 않습니다**. 5분 간 메모리 캐시 후 폐기.

### 3.6 포그라운드 서비스 (`FOREGROUND_SERVICE`, `FOREGROUND_SERVICE_SPECIAL_USE`)
**용도:** 세션 타이머가 OS에 의해 종료되지 않도록 유지.
**수집 정보:** 없음.

### 3.7 정확한 알람 (`SCHEDULE_EXACT_ALARM`, `USE_EXACT_ALARM`) — 선택
**용도:** 예약된 세션을 제시간에 시작 (현재 UI 미노출).
**수집 정보:** 없음.

### 3.8 부팅 완료 수신 (`RECEIVE_BOOT_COMPLETED`)
**용도:** 기기 재부팅 후 진행 중이던 세션 복원.
**수집 정보:** 없음.

### 3.9 배터리 최적화 예외 (`REQUEST_IGNORE_BATTERY_OPTIMIZATIONS`) — 선택
**용도:** 삼성 기기에서 백그라운드 종료 방지.
**수집 정보:** 없음.

## 4. 제3자 공유

**공유하지 않습니다.** 어떤 정보도 제3자 서버·분석 서비스·광고 네트워크로 전송되지 않습니다.

---

## 5. 데이터 삭제

앱을 삭제하거나 "설정 → 앱 → 포커스온 → 저장공간 → 데이터 지우기"를 선택하면 앱이 저장한 모든 정보가 즉시 제거됩니다. 별도의 요청 절차가 필요 없습니다.

---
